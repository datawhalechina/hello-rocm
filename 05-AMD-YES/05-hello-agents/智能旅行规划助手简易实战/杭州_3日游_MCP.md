## 旅行建议
1. **交通推荐**：杭州地铁覆盖主要景点，支持支付宝/杭州通扫码乘车；西湖周边可租借公共自行车（1元/30分钟），环保又便捷。
2. **住宿建议**：第一晚选西湖边民宿，体验“人在画中游”；第二晚可选南宋御街或火车东站附近，方便返程。
3. **避坑提示**：西湖边“游船”推荐正规码头（如湖滨码头、花港观鱼码头），避免黑船；灵隐寺周边小贩拉客需警惕。
4. **文化体验**：可预约“茶文化体验课”（龙井村茶农讲解+亲手炒茶，约120元/人），增强旅行深度。
5. **最佳出行时段**：建议3月–5月或9月–11月，气候舒适，游客适中，拍照出片率高。
