# 🤝 hello-rocm · 组队学习

> **和一群人一起，按节奏把 ROCm 学完。** Datawhale × AMD 官方联合组队学习营，提供免费 GPU 算力、结营证书与布道师直通通道。

## 什么是 hello-rocm 组队学习

hello-rocm 组队学习是 **Datawhale × AMD** 联合发起的线上学习营，围绕本仓库的教程展开。每期聚焦一条学习路径——从环境搭建到大模型部署、微调、再到算子优化——把"自己看文档"升级为"和一群人按节奏共学"。

**你将获得**：

- ☁️ **免费 AMD GPU 算力**：通过 [AMD Radeon Cloud](/zh/cloud/amd-radeon-cloud) 直接上手实操
- 🎓 **官方联合结营证书**：完成全部学习内容即可获得 AMD × Datawhale 双方联合签发的结营证书
- ⭐ **优秀学习者证书**：优质笔记与作品有机会获得优秀学习者证书
- 🚀 **布道师直通**：优秀学习者可直通 **AMD 布道师计划**
- 👥 **共学社群**：同期学习者打卡互助，助教在线答疑
