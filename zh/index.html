推荐学习路径 ​

  1. 先阅读 基础环境，完成 ROCm、PyTorch 与 Python 工具链配置。
  2. 再进入 大模型部署，从 LM Studio 或 vLLM 等推理部署流程开始验证环境。
  3. 环境稳定后继续阅读 大模型微调，了解 ROCm 上的 LoRA 微调流程。
  4. 如果希望深入底层性能与扩展能力，可以继续学习 算子优化。
  5. 需要完整项目样例时，查看 AMD 实践案例。
